MAVEN TO GRADLE - EXP 4
1. Install Java 21
sudo apt update && sudo apt upgrade -y
sudo apt install openjdk-21-jdk -y


java -version
javac -version


Expected Output:
openjdk version "21.x.x"


2. Install Git
sudo apt install git -y
git --version


Fork the repository in GitHub and clone your fork:
git clone https://github.com/<your-username>/simple-java-maven-app.git


Example:
git clone https://github.com/Lucky-Dadlani/simple-java-maven-app.git


3. Install Maven
wget https://archive.apache.org/dist/maven/maven-3/3.9.2/binaries/apache-maven-3.9.2-bin.tar.gz


sudo tar -xvf apache-maven-3.9.2-bin.tar.gz -C /opt


sudo ln -sfn /opt/apache-maven-3.9.2 /opt/maven


echo 'export PATH=/opt/maven/bin:$PATH' >> ~/.bashrc


source ~/.bashrc


mvn -version


Verify that Maven uses Java 21:
mvn -version


Expected Output:
Apache Maven 3.9.2
Java version: 21.x.x


4. Generate Maven Build
cd simple-java-maven-app


mvn clean package


Check build output:
ls target


5. Install Gradle
cd ..


sudo apt install unzip -y


wget https://services.gradle.org/distributions/gradle-8.5-bin.zip


sudo mkdir -p /opt/gradle


sudo unzip -d /opt/gradle gradle-8.5-bin.zip


echo 'export PATH=$PATH:/opt/gradle/gradle-8.5/bin' >> ~/.bashrc


source ~/.bashrc


gradle -v


6. Initialize Gradle Project
cd simple-java-maven-app


gradle init


Select the following options:
Found a Maven build. Generate a Gradle build from this?
yes


Select build script DSL:
2


Generate build using new APIs and behavior?
no


Gradle files will be created:
build.gradle
settings.gradle
gradlew
gradlew.bat
gradle/


7. Edit build.gradle
Open the file:
nano build.gradle (control+x & y then enter to save and exit)


Add the following:
plugins {
    id 'application'
}


Add the following at the end of the file:
jar {
    manifest {
        attributes(
            'Main-Class': 'com.mycompany.app.App'
        )
    }
}


Save and exit.
8. Generate Gradle Build
gradle build


Check the generated JAR:
ls build/libs


9. Execute Gradle Build Output
java -jar build/libs/my-app-1.0-SNAPSHOT.jar


Expected Output:
Hello World!


10. Clean and Rebuild
gradle clean build


11. Execute Maven Build Output
java -jar target/*.jar


Expected Output:
Hello World!
