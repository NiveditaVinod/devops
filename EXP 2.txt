Step 3: Install Gradle
Install unzip:
sudo apt install unzip -y


Download Gradle:
wget https://services.gradle.org/distributions/gradle-8.5-bin.zip


Create Gradle directory:
sudo mkdir /opt/gradle


Extract Gradle:
sudo unzip -d /opt/gradle gradle-8.5-bin.zip


Verify extraction:
ls /opt/gradle


Output:
gradle-8.5


________________


Issue Faced
Command Used
echo "export PATH=/opt/gradle/gradle-8.5/bin:\$PATH" | sudo tee -a /etc/profile.d/gradle.sh


Error
-bash: /etc/profile.d/gradle.sh: No such file or directory


Reason
The environment file could not be created, so Gradle was not added to the system PATH.
When executing:
gradle -v


Error appeared:
gradle: command not found


________________


Fix Applied
Temporarily added Gradle to PATH:
export PATH=/opt/gradle/gradle-8.5/bin:$PATH


Verify:
gradle -v


Output:
Welcome to Gradle 8.5!


Gradle 8.5
JVM: 17
OS: Linux


This confirmed Gradle installation was successful.
