Exp 7: Configuration Management with Ansible – Complete Steps Performed
Step 1: Create Two Ubuntu EC2 Instances
* Master Machine
* Node Machine
Security Group:
* Port 22 (SSH) → Anywhere IPv4
________________


Step 2: Install Ansible on Master
sudo apt update
sudo apt install -y ansible
ansible --version


Verify:
ansible localhost -m ping


Output:
localhost | SUCCESS =>
"ping": "pong"


________________


Mistake 1 ❌
While generating SSH key, you entered:
cd ~/.ssh


at the prompt:
Enter file in which to save the key


Ansible treated it as a filename.
Error:
Saving key "cd ~/.ssh" failed


Fix ✅
Run again:
ssh-keygen


Press Enter for:
Enter file in which to save the key
Enter passphrase
Enter same passphrase


Then verify:
ls ~/.ssh


Output:
id_ed25519
id_ed25519.pub


________________


Step 3: View Public Key
cat ~/.ssh/id_ed25519.pub


Copy the complete output.
________________


Step 4: Login to Node Machine
ssh ubuntu@98.84.112.145


________________


Step 5: Add Public Key to Node
Create SSH folder:
mkdir -p ~/.ssh


Open:
nano ~/.ssh/authorized_keys


Paste public key copied from Master.
Save:
CTRL + O
ENTER
CTRL + X


Set permissions:
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys


________________


Step 6: Test Passwordless SSH
Back on Master:
ssh ubuntu@98.84.112.145


First time output:

Type:
yes


Connected successfully.
Exit:
exit

Step 7: Create Inventory File
nano ~/inventory.ini


Content:
[node_machines]
98.84.112.145 ansible_ssh_user=ubuntu


Save:
CTRL + O
ENTER
CTRL + X


________________


Step 8: Create Playbook
nano ~/install_package.yml


Content:
---
- name: Install a package on the node machine
  hosts: node_machines
  become: yes


  tasks:
    - name: Update apt cache
      apt:
        update_cache: yes


    - name: Install Nginx
      apt:
        name: nginx
        state: present


Save:
CTRL + O
ENTER
CTRL + X


________________


Step 9: Execute Playbook
ansible-playbook -i ~/inventory.ini ~/install_package.yml


Output:
TASK [Update apt cache]
changed


TASK [Install Nginx]
changed


PLAY RECAP
ok=3
changed=2
failed=0


________________
